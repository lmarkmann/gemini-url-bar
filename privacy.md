## Privacy Policy for Gemini URL Prompt

**Last updated:** November 28, 2025

The "Gemini URL Prompt" Chrome extension has been built to improve productivity by prefilling text into the Google Gemini web interface.

**Data Collection**
This extension operates entirely locally on your device. It strictly performs the following actions:
1. Reads the URL parameters of the current tab (specifically on gemini.google.com) to identify the "prompt" text.
2. Injects this text into the page's input field.

We **do not** collect, store, transmit, or share any user data, browsing history, or personal information with the developer or any third parties. No data leaves your computer.

**Contact**
If you have any questions about this privacy policy, please contact us via the Chrome Web Store support page.
